# How to run
 
- `git clone https://github.com/ashraf-alsamman/webconia-frontend`
- Edit config/confige.js api  
- Run frontend (react js) project
  `npm install`
  `npm start`
  
  - Run tests
  `npm test`

Try localhost/3000
